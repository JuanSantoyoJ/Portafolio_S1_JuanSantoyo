Sobre Mí

Soy un desarrollador Full Stack con pasión por la tecnología y la creación de soluciones innovadoras. A los 18 años, estoy en constante aprendizaje, explorando nuevas herramientas y lenguajes para construir experiencias digitales robustas y atractivas. Mi enfoque se centra en el desarrollo de aplicaciones web eficientes, funcionales y con un diseño moderno.

Mi objetivo es seguir creciendo profesionalmente, colaborando en proyectos que me permitan aplicar y expandir mis conocimientos en el desarrollo web.

Proyectos

Formula 1

Desarrollé una aplicación web completa para gestionar de manera eficiente los datos de la Fórmula 1. Este proyecto CRUD (Crear, Leer, Actualizar, Eliminar) permite a los usuarios mantener una base de datos actualizada con la información de pilotos, escuderías y sus estadísticas. La aplicación fue construida utilizando HTML para la estructura, CSS para el diseño visual y JavaScript para la lógica del frontend, garantizando una experiencia de usuario fluida e interactiva.

Gestion Hospitalaria

El proyecto incluye el modelado de la base de datos para manejar la información de hospitales, pacientes, médicos y tratamientos. También especifica la implementación de consultas, la creación de funciones simuladas y la definición de roles de usuario con permisos específicos. En resumen, el documento detalla los requerimientos para un proyecto académico de base de datos no relacional que gestiona todas las operaciones de un sistema hospitalario.
